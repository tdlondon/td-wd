when do we use tables

  • to display tabular data
  • to create html emails (marketing, newsletter etc)
  • we do not use them for layouts anymore

table tag and attributes

  • table tag
  • width fixed or fluid
  • border in px
  • align left / centre / right
  • cellpadding to control spacing inside cells
  • cellspacing to control spacing in between cells

<table width="250" border="1" align="center"
    cellpadding="20" cellspacing="2">
    <!-- ... -->
</table>
  • DEMO: table tag markup

inserting rows

  • count and insert rows
  • Table Row tags tr

<table width="250" border="1" align="center"
        cellpadding="20" cellspacing="2">
    <tr></tr>
    <tr></tr>
</table>
  • DEMO: a table with two rows

inserting cells

  • Table Headings tags th
  • regular cells - Table Data tags td
  • cellspacing controls space in between cells
  • cellpadding controls space inside cells

<table border="1" width="250" border="1" align="center"
        cellpadding="20" cellspacing="2">
    <tr>
        <th>H</th>
        <th>H</th>
    </tr>

    <tr>
        <td>A</td>
        <td>B</td>
    </tr>
</table>
  • DEMO: table headings and table cells

merging cells

  • we can merge cells horizontally with colspan attribute
  • we can merge cells vertically with rowspan attribute

<table align="center" border="1" width="450" border="1"
       cellpadding="20" cellspacing="2">
    <tr>
        <th colspan="2">colspan=2</th>
    </tr>
    <tr>
        <td rowspan="2">rowspan=2</td>
        <td>A</td>
    </tr>
    <tr>
        <td>B</td>
    </tr>
    <tr>
        <td>C</td>
        <td>D</td>
    </tr>
</table>
  • DEMO: a table with merged cells
