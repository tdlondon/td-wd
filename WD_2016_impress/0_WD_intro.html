requisites

To create websites you will need:

A PC or a Mac

All technologies that we will cover are CROSS PLATFORM, you will be able to use any OS (Windows, MAC OSX, Linux)

A Web Editor / IDE

  • a Web Editor is a software that recognises web languages
  • IDE stands for Integrated Development Environment (web editor, build automation tools, debuggers)
  • there are many web editors, some are cheap or free, others are expensive

Brackets

  • http://brackets.io
  • open source
  • live preview
  • easy to extend
  • cross platform

Aptana Studio

  • http://www.aptana.com/
  • open source
  • supports many languages
  • solid and reliable
  • cross platform

WebStorm

  • https://www.jetbrains.com/webstorm/
  • cheap
  • supports many languages
  • very good for client side languages
  • cross platform

Sublime Text

  • http://www.sublimetext.com/
  • cheap
  • supports many languages
  • very flexible, plenty of plugins
  • cross platform

Dreamweaver

  • http://www.adobe.com/uk/products/dreamweaver.html
  • expensive
  • code and design panel
  • cross platform
