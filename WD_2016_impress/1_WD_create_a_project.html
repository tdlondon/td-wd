Starting a project

  • Brackets project
  • WebStorm project

Brackets

  • download from http://brackets.io/
  • launch Brackets
  • create a folder on your desktop
  • drag it into Brackets
  • right-click to create files / folders

WebStorm

  • download from https://www.jetbrains.com/webstorm/download/
  • launch WebStorm
  • click on "new project"
  • select path and eventually create a folder
  • right-click to create files / folders
